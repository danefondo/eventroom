# elysiumbase

# installation
If using Windows, comment out 'postinstall' line in package.json 
npm install
If using Windows, cd to client, then 
npm install
npm install --only=dev
npm run build
if problem with node-sass
npm rebuild node-sass
npm run build 
Then cd .. back and run 
npm run dev
